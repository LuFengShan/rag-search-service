## 基本信息

丰田亚洲龙2025款定位B+级轿车，基于TNGA-K平台，车身尺寸：4990×1850×1450mm，轴距2870mm。有2.0L燃油和2.5L双擎两个版本。

## 配置与价格

| 配置版本 | 指导价 | 动力 |
|----------|--------|------|
| 2.0L 进取版 | 19.98万 | 2.0L+CVT |
| 2.0L 豪华版 | 21.78万 | 2.0L+CVT |
| 2.5L 双擎 豪华版 | 23.98万 | 2.5L混动 |
| 2.5L 双擎 尊贵版 | 25.78万 | 2.5L混动 |
| 2.5L 双擎 旗舰版 | 27.98万 | 2.5L混动 |

## 动力与油耗

2.0L版最大功率130kW，WLTC油耗6.0L。2.5L双擎版系统综合功率160kW，WLTC油耗4.7L。

## 核心卖点

1. **越级空间**：车长近5米，后排空间媲美豪华品牌
2. **美式舒适**：悬架调校偏舒适，长途不累
3. **混动油耗**：双擎版百公里不到5L油
4. **商务气场**：外观大气稳重，商务接待有面子
5. **静谧性**：车内隔音出色，高速也很安静

## 适合人群

- 预算20-28万的商务人士
- 经常跑长途、注重乘坐舒适性的用户
- 追求大空间轿车的家庭
- 对品牌认可度高、不想买豪华品牌但想要品质的用户

## 竞品对比

vs 大众迈腾：亚洲龙空间更大、混动油耗更低，迈腾操控略好
vs 比亚迪汉：亚洲龙品牌溢价高，保值率优势明显，但汉的电动化体验更好
